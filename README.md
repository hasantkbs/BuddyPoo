# BuddyPoo

BuddyPoo is an AI-powered storytelling tool for children. It generates creative and engaging stories to spark imagination and make reading fun.

## Features

*   **AI-powered story generation:** Creates unique stories based on user prompts.
*   **Interactive storytelling:** Allows children to influence the story's direction.
*   **Safe and age-appropriate content:** Ensures all stories are suitable for children.
